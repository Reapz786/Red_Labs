# Resume/CV Setup Instructions

## What You Need To Do:

### 1. Clean Your CV
Open `EhsanCVSite.docx` and make these changes:

**Remove:**
- "Mobile: (upon request)" line

**Keep everything else** - your CV looks great!

### 2. Export to PDF

**Option A: Microsoft Word**
1. File → Save As
2. Choose "PDF" format
3. Save as: `Ehsan_Ali_Resume.pdf`

**Option B: Google Docs**
1. Upload DOCX to Google Drive
2. Open in Google Docs
3. File → Download → PDF
4. Save as: `Ehsan_Ali_Resume.pdf`

**Option C: Online Converter**
1. Go to: https://www.ilovepdf.com/word_to_pdf
2. Upload your DOCX
3. Convert to PDF
4. Download as: `Ehsan_Ali_Resume.pdf`

### 3. Add to Portfolio

1. Place `Ehsan_Ali_Resume.pdf` in: `assets/resume/`
2. Push to GitHub:
   ```bash
   git add assets/resume/Ehsan_Ali_Resume.pdf
   git commit -m "Add resume PDF"
   git push
   ```

### 4. Test

Visit your site and click "DOWNLOAD RESUME/CV" button!
